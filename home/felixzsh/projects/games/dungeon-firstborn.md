# Dungeon Firstborn

Dungeon Firstborn is a turn-based dungeon crawler game developed for a game jam, where it achieved 27th place from more than 100 games. This project marks my first participation in a game jam and showcases fundamental dungeon crawler mechanics.

## Key Features:

*   Turn-based combat system
*   Procedural level generation
*   Melee and ranged combat options
*   Item system (e.g., potions)
