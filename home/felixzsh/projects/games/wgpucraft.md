# Wgpu Minecraft Clone

This project is a Minecraft clone developed from scratch using the `wgpu` graphics library and the Rust programming language. It's a hobby project to explore game development fundamentals and low-level graphics APIs.

## Key Features:

*   Atlas texture implementation
*   Efficient block rendering with face culling
*   First-person controller
*   Procedural terrain generation using noise maps
*   Chunk culling
